basicHTML

Simplified DOM Back End

Similar to jsdom only on the surface, basicHTML is a lightweight, dumb, and simple, implementation of the HTML standard.
It's the project that made it possible to define via NodeJS, NativeScript Custom Elements such <Page.actionBar> and much more.

  • create any amount of documents you want
  • document fragments, comments, text nodes, and elements
  • case sensitive and compatible with XML structures
  • Custom Elements, with observable attributes
  • Event Listener, dispatcher, DOM Level 0, classList, and more

Any transformation from HTML to any platform that supports JavaScript is now easy to create and use, thanks to basicHTML sharable Custom Elements definition and the magic of hyperHTML.
